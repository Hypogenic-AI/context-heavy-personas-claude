# Context Heavy Personas

**Research Question**: Do certain LLM personas require extensive in-context conditioning (10K+ tokens) to emerge? If not, what does this imply about the total number of personas an LLM can represent?

## Key Findings

- **No context-heavy personas found**: All 25 tested personas from the Anthropic Model-Written Evals benchmark achieve ≥95% accuracy at K=0 (zero-shot) on GPT-4.1-mini
- **Binary agreement saturates immediately**: The Kruskal-Wallis test shows no significant effect of context length on accuracy (p=0.44)
- **Generation quality shows marginal improvement**: LLM-judged persona distinctiveness improves by +0.18/10 with more context (p=0.038), driven primarily by "politically-liberal" and "self-replication" personas
- **The persona space is "shallow but vast"**: For frontier models, all tested personas are specifiable with short prompts, suggesting the accessible persona count is bounded by the combinatorial space of concise descriptions, not by context-length activation thresholds
- **Only safety-constrained personas benefit from context**: The few personas showing any K=0 difficulty are AI-risk-related (no-shut-down, self-replication, escape-sandbox), where RLHF resistance is overcome by demonstrations
